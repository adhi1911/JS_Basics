/* All events run sequentially exceptions are asynchronous events 1. Event Bubbling: The event starts from the target element and bubbles up to the root element. 2. Event Capturing: The event starts from the root element and goes down to the target element. 3. Event Delegation: A technique where a single event listener is added to a parent element to manage events for multiple child elements. 4. Event Propagation: The process of how events travel through the DOM tree, either bubbling or capturing. 5. Event Target: The element that triggered the event. 6.
